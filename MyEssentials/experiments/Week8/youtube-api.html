Experiment with Youtube API

In this experiment, we experiment on Youtube API. We can search videos on youtube by entering the search term in the input box.

Code Documentation

function search(query) {
var request;
request = gapi.client.youtube.search.list({
part: 'snippet',
maxResults: 10,
q: query
});
request.execute(showResponse);
}
