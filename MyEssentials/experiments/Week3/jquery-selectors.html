jQuery Selectors

jQuery selectors can be used to select and manipulate HTML element(s).
The JQuery function understands if the input given is a String or a JSON Object. In this way, we could programmatically manipulate any element in the HTML DOM Structure
The JQuery function can be called by using a dollar "$" sign.
All selectors in jQuery start with the dollar sign and parentheses: $().
