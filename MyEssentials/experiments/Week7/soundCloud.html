Using SoundCloud API

In this experiment, We use soundCloud API to get songs of some specific genres and play them on our webpage.

  • dubstep
  • electronic
  • punk

Code Documentation

function playSomeSound(genre) {
SC.get('/tracks', {
genres: genre,
bpm : {
from: 100
}
}, function(tracks){
var random = Math.floor(Math.random() * 49);
SC.oEmbed(tracks[random].uri, {autoplay: true}, document.getElementById('target'));
});
}
