Experiment with Youtube API with Scrollable results

In this experiment, we continue with the previous experiment for Youtube API. We can search videos on youtube by entering the search term in the input box. In addition to this, we create a scroll bar to display the results

Code Documentation

function search(query) {
var request;
request = gapi.client.youtube.search.list({
part: 'snippet',
maxResults: 10,
q: query
});
request.execute(showResponse);
}
