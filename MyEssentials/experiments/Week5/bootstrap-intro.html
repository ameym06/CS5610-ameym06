Bootstrap is a free collection of tools for creating websites and web applications.
It contains HTML and CSS-based design templates for typography, forms, buttons, navigation
and other interface components, as well as optional JavaScript extensions.
Most common use of the Bootstrap is to decorate the Website so as it looks attractive
and give it an artistic look.
