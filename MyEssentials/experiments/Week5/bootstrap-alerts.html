The Alerts usually provide contextual feedback messages 
for typical user actions with the handful of available and flexible alert messages.
Simply Wrap any text and an optional dismiss button in .alert and one of the 
four contextual classes (e.g., .alert-success, .alert-warning, .alert-info etc.) for basic alert messages.
