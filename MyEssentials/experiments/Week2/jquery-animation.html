Element Animations

Description

The jQuery animate() method has been utilized to create division and text based animations. A click on the 'Animate' button starts the sequence. First of all, the text changes, followed by a change in width, height, opacity level and text size. The animation sequence continues with a subsequent reduction in height and width respectively. This would be the final state of our division tag. Another click on the animate button starts the process over. However, the text and opacity level do not change.

jQuery Animations

The 'animate' method has been used to apply animation to an entire division tag. A mouse click event on 'Animate' button executes an animation sequences targeted on 'div' with class specified as 'contentBox'. The first call to animate expands contentBox's width, followed by its height. It then proceeds to change the division's opacity level and font size. Finally, the height and width are reduced to original levels. A change in text from 'Black' to 'Gray' also takes place.

$(".animateBtn").click(function()
{
var div = $(".contentBox");

div.animate({ width: "400px" }, "slow");
div.animate({ height: "400px" }, "slow");
div.animate({ opacity: "0.5" }, "slow");
div.animate({ fontSize: "6em" }, "slow");
div.animate({ height: "500px" }, "slow");
div.animate({ width: "200px" }, "slow");
div.text("Gray");
});
