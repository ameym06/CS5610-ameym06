Slide effect

It is not safe to manipulate a page until the time it has finished loading. The 'ready' function makes this check to verify when the Document Object Model (DOM) is ready for JavaScript code to safely execute. 'click' is used to detect click events on the selected element, 'hide' does what its name suggests and 'slideDown' slides the mentioned elements. The form used here takes speed as an argument.

$(document).ready(function()
{
$(".flipBox").click(function()
{
$(this).hide();
$(".contentBox").slideDown(5000);
});
});
