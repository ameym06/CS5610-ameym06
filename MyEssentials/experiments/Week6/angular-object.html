Angular Objects

Documentations

In this experiment, we are going to see the creations and usage of objects in angular js. Objects are just like the sturctures that are defined in the C++, or JAVA. We can store variables, media, images or other objects into a single object. These values than can be formulated and used in the page itself.

Example1

In this example we are just going to initialise the object "book" that has the following fields
"Title"
"Author"
"Price"
these field are initialized into the ng-init directives as follows
ng-init="book={title:'Web Development Fundamentals', author:'Prof. Jose', price:'120'}"
Finally we print these values using a simple "ng-bind" directives.

function bookController($scope) {
$scope.book = {
title: "Web Development Fundamental",
author: "Professor Jose",
price: "120",
fullInfo: function () {
var x;
x = $scope.book;
return "Title: " + x.title + " Author:" + x.author + " Price: $" + x.price;
}
};
}
