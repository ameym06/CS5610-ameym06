View Blog Functionality

This Page can be used only by registered users who have logged into the application. This Page can be used by the users to view their own blog in details. When the user selects a particular Blog from the 'MyBlogs' page, then the select blog is rendered onto this page so that the user can read the blog in detail.


Snapshot

View Blog Services

View Blog services are designed using Angular.JS

  1. View Blogs Service - Allows the logged in user to view their blog in detail. During the page load, AngularJS stores the data from the 'MyBlogs' page and renders the selected blog onto this page.
